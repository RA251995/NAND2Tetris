#### Arithmetic / Logical Commands
command | operation | returns
--------|-----------|---------
`add`   | x+y       | integer
`sub`   | x-y       | integer
`neg`   | -y        | integer
`eq`    | x==y      | boolean
`gt`    | x>y       | boolean
`lt`    | x<y       | boolean
`and`   | x AND y   | boolean
`or`    | x OR y    | boolean
`not`   | NOT y     | boolean
